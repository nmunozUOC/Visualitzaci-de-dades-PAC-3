# Visualització de dades – PAC 3

Aquest repositori correspon a la **PAC 3 de l'assignatura Visualització de Dades**.

## Descripció del projecte

El projecte consisteix en el desenvolupament d’una **visualització de dades interactiva**, implementada mitjançant tecnologies web. L'objectiu és analitzar i comunicar informació de manera clara i efectiva a través de gràfics i elements visuals.

## Estructura del repositori

- `code/`  
  Conté el codi font necessari per generar la visualització

- `Hotel_Booking_Story (11).html`  
  Fitxer HTML principal amb la visualització final del projecte.

- `README.md`  
  Documentació del projecte.
